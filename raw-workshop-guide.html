应用介绍

什么是 RAW工坊？

热爱摄影的开发者，为摄影爱好者打造专业的 RAW 图像处理工具

RAW工坊（原名 RapidRAW）是一款现代高性能的 RAW 图像编辑器，定位为 Adobe Lightroom 的轻量替代方案。它采用 Rust + React + WGSL GPU 着色器 构建，将完整的 32 位浮点图像处理管线卸载到 GPU，实现拖动滑块时的即时反馈。

应用遵循非破坏性工作流——所有编辑参数保存在独立的 .rrdata 附属文件中，原始图像始终保持不变。无论你调整曝光、应用蒙版还是裁剪变换，都可以随时撤销或重新编辑。

从 v1.6.1 起，RAW工坊引入了深空黑 + 青墨绿的默认主题色，并以中文为原生界面语言，面板交互按照国内摄影师的习惯重新排列：基础 → 色彩 → 人像 → 构图 → 蒙版 → AI → 预设 → 导出。

💡
核心架构：Rust 后端负责 RAW 解码、文件管理和导出；WGSL 着色器在 GPU 上完成所有像素级处理；React + Tauri 提供流畅的跨平台 UI。三者协作，实现从图库浏览到实时编辑的全链路高性能体验。
